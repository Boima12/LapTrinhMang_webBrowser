# Lập trình mạng - Web browser

## Cách chạy

1. vào src/server/ chạy file SimpleHttpServer.java để bật server
2. vào src/client/ chạy file HttpFxClient.java để mở client browser (thông tin phụ: client browser có thể chạy mà không cần mở server, tuy nhiên lại không thể chạy các đường url như http://localhost:8080/index.html)
